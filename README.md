This project focuses on optimizing inventory in a supply chain by analyzing demand patterns, supplier performance, and stock levels to reduce overstock and stockouts.

🎯 Problem Statement
Overstock leads to increased holding cost
Stockouts lead to lost sales
Need to optimize inventory using data-driven insights

🛠️ Tools Used
Python (Pandas, Matplotlib, Forecasting)
SQL (Data extraction from ERP dataset)
Power BI (Dashboard & Reporting)

📂 Project Workflow
Data Extraction using SQL
Data Cleaning & EDA using Python
Demand Forecasting (Time Series)
Risk Analysis (Stockout & Overstock)
Dashboard creation in Power BI

📈 Key Insights
Identified fluctuating demand patterns
Forecasted future demand using time series
Detected stockout and overstock risks
Analyzed supplier delays impacting inventory



🚀 Outcome
Reduced inventory wastage by identifying overstock and stockout risks  
Optimized inventory planning using demand forecasting, reducing holding costs  
Enabled data-driven decision-making for efficient supply chain management
